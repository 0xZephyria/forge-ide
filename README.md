# FORGE WEB IDE

Forge is a high-performance, safety-first smart contract language designed for Zephyria Network's Native Contract Language and the next generation of decentralized applications. Built with Zig, Forge compiles to high-efficiency RISC-V bytecode optimized for Zephyria ZVM, PolkaVM & EVM (Under Development).
